City Life

*Insert An Actual GOOD Name for this Application* is useful for finding a list of the top 10 most popular restaurants in a given city. Works well for bigger cities, smaller cities just don't contain the same useful information and picture availability.


This application was created using JavaScript and styled with custom CSS. The information for restaurants is provided by Zomato and the pictures of the cities are provided by Pixabay.


I created this to learn how to load information from API's asynchronously onto the page and display it nicely. Ideally want to find more information to load for cities but not sure what API's to use and whether it even makes sense.
